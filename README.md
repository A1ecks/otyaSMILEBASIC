# How to build
```sh
git clone --recursive https://github.com/otya128/otyaSMILEBASIC.git
cd otyaSMILEBASIC
dub run
```

## Dependency:build
+ D compiler
+ dub
+ DerelictSDL2
+ DerelictGL3
+ curl

## Dependency:run
+ SDL2
