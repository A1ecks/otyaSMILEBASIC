# ビルド方法
```sh
git clone --recursive https://github.com/otya128/otyaSMILEBASIC.git
cd otyaSMILEBASIC
dub run
```

## コンパイルするのに必要なもの
+ Dコンパイラ
+ dub
+ DerelictSDL2
+ DerelictGL3
+ curl

## 実行時に必要なもの
+ SDL2
